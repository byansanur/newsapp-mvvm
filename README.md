# newsapp-mvvm
Simple app for learning mvvm architecture.
Visit link to learn Design Pattren [MVVM](https://medium.com/@syafdia/mengenal-arsitektur-model-view-viewmodel-mvvm-di-android-2e52ec98a74e)

# Project resource
- Retrofit [visit link to documentation](https://square.github.io/retrofit/)
- OkHttp [visit link to documentation](https://square.github.io/okhttp/3.x/okhttp/index.html?okhttp3/OkHttpClient.html)
- Coroutines [visit link to documentation](https://kotlinlang.org/docs/reference/coroutines-overview.html)
- LiveCycle ViewModel [visit link to documentation](https://developer.android.com/jetpack/androidx/releases/lifecycle)
- Room [visit link to documentation](https://developer.android.com/topic/libraries/architecture/room?gclid=CjwKCAjw8J32BRBCEiwApQEKgcPiBKCYHmkI1GxBX6mO0ZzWwMGF--HuHiUa-Mbnhdf1EVqbT0TVGBoCLCoQAvD_BwE&gclsrc=aw.ds)

# Project setUp
- Install Android studio current version [Android studio](https://developer.android.com/studio/?gclid=CjwKCAjw8J32BRBCEiwApQEKgfuMVBCRe8J3K_E5ZiF8tckadHmzny33YBAiUFYenf1PbzG7TfevAxoCOm8QAvD_BwE&gclsrc=aw.ds)
- Install [JVM](https://www.java.com/en/download/)
- Install [kotlin plugin in android studio](https://developer.android.com/kotlin/get-started?gclid=CjwKCAjw8J32BRBCEiwApQEKgSfqQKLKdr4iAhBhdyeliOkd5qo6Oo_-jTSg6CTTkHgx3VX6AL8UvxoCMU0QAvD_BwE&gclsrc=aw.ds)
- Install [gradle](https://gradle.org/install/)
- Clone [repository](https://www.londonappdeveloper.com/how-to-clone-a-github-project-on-android-studio/)
- Run [App emulator](https://developer.android.com/studio/run/emulator)
